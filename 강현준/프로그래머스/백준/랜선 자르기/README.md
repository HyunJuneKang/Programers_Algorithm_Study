랜선 자르기
문제 개념

이미 가지고 있는 랜선 K개가 있고, 각 랜선의 길이는 제각각이다.

이 랜선들을 모두 같은 길이로 잘라서 총 N개 이상의 랜선을 만들고 싶다.

이때 만들 수 있는 랜선의 최대 길이를 구하는 문제이다.

예를 들어 랜선 길이가 다음과 같고,

802 743 457 539

필요한 랜선 개수가 11개라면, 길이를 200으로 잘랐을 때:

802 / 200 = 4개
743 / 200 = 3개
457 / 200 = 2개
539 / 200 = 2개

총

4 + 3 + 2 + 2 = 11개
